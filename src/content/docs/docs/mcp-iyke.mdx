`mcp-iyke` is a thin MCP server that wraps the Iyke runtime bridge — the localhost JSON-RPC surface exposed by a running shell. With it installed, any MCP-capable model (Claude, Cursor, custom clients) can navigate panes, switch modes, inspect shell state, and read accessibility trees.

It pairs with the **`iyke`** Rust CLI, which speaks the same bridge for terminal-based control.
