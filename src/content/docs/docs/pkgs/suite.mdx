**Suite** (`@ikenga/pkg-suite`) is a business suite that runs as a panel inside the Ikenga shell — Sales, Outbound, and Email views in one place. It's a multi-file iframe pkg with no build step, backed by Supabase.

## Install

```bash
ikenga add @ikenga/pkg-suite
```

Requires the Ikenga shell. [Install the shell →](/#install)

## What it is

| | |
| --- | --- |
| **Package** | `@ikenga/pkg-suite` (`com.ikenga.suite`) |
| **Archetype** | UI iframe (`kind: embedded`) — a panel in the shell's **Ops** section |
| **Backend** | Supabase (required), tables `deals`, `outbound_messages`, `email_drafts` |
| **Network** | `https://esm.sh` (module CDN) + your Supabase project |

## Supabase credentials

Suite declares the **Supabase capability** rather than baking keys into the build. The shell resolves credentials from its encrypted vault and threads them through the AppBridge handshake at mount time — so the keys never live in the package. (See [Pkgs → Capabilities](/docs/pkgs).)
